ArrayTable

ArrayTable 介绍：优雅实现表格自增等功能组件#

属性#

属性说明默认值
tableConfig?: TablePropstable 的配置，即 antd 的 Table 的配置，当 render 中有 FormItem 时，将自动添加数据监听和映射
value?: any[]值
onChange?: (data: any[]) => void内部 FormItem 的 onChange 触发时触发
rowKey?: string数据唯一的标识'key'
onRowChange?: (val: any, core: Core) => void每行的 onChange
bottom?: ReactNode列表底部内容
top?: ReactNode列表顶部内容
rowFormConfig?: FormProps每行 Form 的 props 配置
rowCoreConfig?: CoreProps每行 Core 的构建参数
className?: string根节点类名
style?: React.CSSProperties根节点 style
