Form 组件介绍#

属性#

属性说明默认值
core?: Core表单核心
colon?: booleanFormItem label 后是否带 ":"false
onChange?: (val: any, core: Core) => voidonChange 事件，触发时机即 Core 的 onChange
value?: any表单值，当变化时，会更新 core 的值
inline?: booleanFormItem 行内展示false
full?: booleanFormItem 撑满false
direction?: "vertical" | "horizontal"FormItem label 和组件水平或垂直排列'vertical'
status?: StatusFormItem 的 status'edit'
Com?: HTML element根节点 Dom'div'
className?: string根节点 className
style?: React.CSSProperties根节点 style
onMount?: (core: Core) => voidForm ComponentDidMount 触发
globalStatus?: Status全局 status'edit'

技巧和特性#

Form 支持嵌套#

  • 内部 Form 的状态基于最近的 FormItem 的状态
  • 内部 Form 将继承外部 Form 的 colon,inline,direction,full 属性
<Form
value={{
inner: {
innerItem: "init",
innerInner: {
innerInnerItem: "init",
innerInnerInner: { innerInnerInnerItem: "init" },
},
},
}}
colon={true}
>
<FormItem name="inner" status="disabled">
<Form>
<FormItem name="innerItem" label="innerItem">
<Input />
</FormItem>
<FormItem name="innerInner" status="edit">
<Form>
<FormItem name="innerInnerItem" label="innerInnerItem">
<Input />
</FormItem>
<FormItem name="innerInnerInner">
<Form>
<FormItem name="innerInnerInnerItem" label="innerInnerInnerItem">
<Input />
</FormItem>
<FormItem
name="innerInnerInnerItem1"
label="innerInnerInnerItem1"
>
<Input />
</FormItem>
</Form>
</FormItem>
</Form>
</FormItem>
</Form>
</FormItem>
</Form>
